## API for data like smartcity.jakarta.go.id

### Setup

Configure your Heroku https://dashboard.heroku.com

Run **npm install** to get all required libraries.

	npm install

### Base URL

	api baseURI is at /api/

	ex:
	<base_url>/api/haltebus

### Endpoint

- haltebus
- kepolisian
- rumahsakit
- sekolah
- lokasitransportasi
- tempatibadah
